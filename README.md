# 配送月清单导入工具

本工具用于把客户订单导出表导入到学校配送月清单模板中，并生成新的月份 sheet。

## 启动

```powershell
python app.py --host 127.0.0.1 --port 8765
```

打开：

```text
http://127.0.0.1:8765
```

## 处理规则

- 订单导出文件可以多选批量上传，每个文件需要包含：`订单号`、`客户名称`、`发货日期`、`商品名称`、`商品描述`、`实际数量`、`发货单位`、`发货单价`、`实际金额`。
- 目标学校月清单模板可以多选批量上传，每个模板会自动读取 `配送学校：...` 作为学校名。
- 普通营养餐匹配源文件中的同名学校。
- 早晚餐匹配源文件中的 `学校名-早晚餐`。
- 按发货日期所在月份生成 sheet，例如 `2026年6月`、`2026年6月 早晚餐`。
- 新 sheet 会复制目标模板中同类型最近月份的格式，再写入明细、日小计和月合计。
- 默认不覆盖已有同名 sheet；如果同名 sheet 已存在，会生成带 `导入` 后缀的新 sheet。
- 首页会显示已生成的目标清单，按更新时间倒序排列，最新文件在最上方。
- 生成结果保存在 `outputs/`，不会自动删除；只有在首页点击删除并确认后才会移除。
- 批量生成后，每个目标学校会得到一个独立结果文件，同时会生成一个 ZIP 用于一键下载全部结果。
- 结果页提供“选择文件夹保存”按钮；该能力依赖浏览器支持文件夹写入，若当前浏览器或 HTTP 访问方式不支持，请使用 ZIP 下载。

## 命令行处理

```powershell
python app.py --source "订单导出.xlsx" --template "学校月清单.xlsx" --output "导入结果.xlsx"
```
