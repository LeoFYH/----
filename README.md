# 配送月清单导入工具

本工具用于把客户订单导出表导入到学校配送月清单模板中，并生成新的月份 sheet。

## 启动

```powershell
python app.py --host 127.0.0.1 --port 8765
```

打开：

```text
http://127.0.0.1:8765
```

## 处理规则

- 订单导出文件需要包含：`订单号`、`客户名称`、`发货日期`、`商品名称`、`商品描述`、`实际数量`、`发货单位`、`发货单价`、`实际金额`。
- 目标模板会自动读取 `配送学校：...` 作为学校名。
- 普通营养餐匹配源文件中的同名学校。
- 早晚餐匹配源文件中的 `学校名-早晚餐`。
- 按发货日期所在月份生成 sheet，例如 `2026年6月`、`2026年6月 早晚餐`。
- 新 sheet 会复制目标模板中同类型最近月份的格式，再写入明细、日小计和月合计。
- 默认不覆盖已有同名 sheet；如果同名 sheet 已存在，会生成带 `导入` 后缀的新 sheet。

## 命令行处理

```powershell
python app.py --source "订单导出.xlsx" --template "学校月清单.xlsx" --output "导入结果.xlsx"
```
